# Hype

Автоматизированная CPA-платформа для заработка на рекомендациях топ-брендов.

Зарабатывай на действиях пользователей: покупки, заявки, регистрации. Получай повышенные выплаты до 80% при регистрации по промокоду.

---

## Проблема

- CPA-сети есть, но без партнёрской структуры — зарабатывает только верхушка
- Амбассадоры не могут строить команду внутри существующих CPA-сетей
- Рекламодатели тратят миллионы на агентства-посредники
- Обычный пользователь не знает, как монетизировать свой трафик

## Решение

Hype — CPA-платформа с партнёрской структурой:

- **Представитель** — продвигает CPA-офферы, получает оплату за действие (покупка, заявка, регистрация)
- **Амбассадор** — строит команду представителей, получает процент с их конверсий
- **Основатель** — управляет платформой, подключает рекламодателей, настраивает комиссии

---

## 1. Роли в системе

### Представитель (Дистрибьютор)
- Регистрируется по промокоду
- Получает доступ к каталогу CPA-офферов
- Зарабатывает на каждом целевом действии пользователя
- Может выдавать свои промокоды новым участникам

### Амбассадор (Партнёр)
- Создаёт и распространяет промокоды
- Получает процент с конверсий представителей, зарегистрированных по его промокоду
- Имеет доступ к дашборду команды
- Выходит на этот уровень по приглашению или после KPI (5+ активных представителей, каждый получил выплаты более двух раз)

### Основатель (Администрация)
- Управляет платформой
- Подключает рекламодателей и офферы
- Устанавливает правила и комиссии
- Получает процент со всей системы

---

## 2. Система промокодов

### Как это работает
- Каждый представитель и амбассадор имеет уникальный промокод
- Промокод вводится при регистрации нового участника
- После ввода промокода участник навсегда привязывается к тому, кто выдал промокод

### Типы промокодов

| Тип | Кто выдаёт | Для кого | Эффект |
|---|---|---|---|
| **Промокод амбассадора** | Амбассадор | Новые представители | Представитель получает повышенные выплаты (80%), амбассадор — 10% |
| **Промокод представителя** | Представитель | Новые представители | Представитель становится амбассадором (накапливает команду) |
| **Поиск промокода** | Любой | Все | Введите в поисковике `hype-XXXX-2026` и возьмите любой промокод |

### Формат промокода
- `HYPE-XXXXXXX` — стандартный формат
- Уникальный, не может быть изменён
- Действует бессрочно

### Как получить промокод

Если у вас нет промокода от знакомого:

1. Зайдите в любой поисковик (Яндекс, Google)
2. Введите запрос: `hype-XXXX-2026`
3. Возьмите первый попавшийся промокод из результатов
4. Зарегистрируйтесь с ним

Все промокоды равнозначны — каждый открывает повышенные выплаты 80%.

Регистрация без промокода недоступна. Каждый участник должен ввести промокод при регистрации.

## 3. Распределение вознаграждений

### Базовая схема (с промокодом)

```
Рекламодатель платит:               100% (например, 1 000₽)
  ├── Представитель                  80%  (800₽)
  ├── Амбассадор (кто выдал промокод) 10% (100₽)
  └── Основатели (платформа)         10% (100₽)
```

### Если представитель сам стал амбассадором

Если представитель A привёл представителя B:
- A получает 10% с конверсий B (как амбассадор по отношению к B)
- A продолжает получать 80% со своих собственных конверсий
- Первый амбассадор (кто выдал промокод A) получает 10% с конверсий A, но **не получает** % с конверсий B

### Максимальная глубина
- Система поддерживает **2 уровня глубины**:
  - Уровень 1: амбассадор → представитель
  - Уровень 2: представитель → новый представитель (бывший представитель становится амбассадором)
- Дальнейшая глубина не предусмотрена

## 4. Структура команды

```
Амбассадор
  ├── Представитель A (по промокоду амбассадора)
  │     └── Представитель B (по промокоду A) → A получает 10% от B
  ├── Представитель C (по промокоду амбассадора)
  └── Представитель D (по промокоду амбассадора)
```

Правила:
- Амбассадор получает 10% от конверсий A, C, D
- Представитель A получает 10% от конверсий B
- Амбассадор **не получает** % от B (только свой прямой уровень)
- Каждый представитель в любой момент может начать приглашать других и получать свой %

## 5. Офферы и выплаты

### Типы офферов

| Тип | Описание | Пример | Комиссия представителю |
|---|---|---|---|
| **CPA (фикс)** | Фиксированная сумма за действие | Заявка на карту Халва — 3 000₽ | 80% от ставки = 2 400₽ |
| **CPS (%)** | Процент от продажи | Wildberries — до 34% | 80% от комиссии = до 27.2% |
| **Реферал** | Вознаграждение за приглашённого | Яндекс Маркет — 15 000₽ за продавца | 80% = 12 000₽ |

### Каталог CPA-офферов

| Рекламодатель | Действие | Ставка | Канал |
|---|---|---|---|
| **Wildberries / Ozon** | Покупка товара | CPS до 34% | ТакПродам |
| **Халва (Совкомбанк)** | Заявка на карту рассрочки | CPA до 3 000₽ | Affilead / Admitad |
| **Яндекс Маркет** | Реферал нового продавца | CPA 15 000₽ | Прямая программа |
| **Тинькофф Банк** | Заявка на карту / кредит | CPA до 3 000₽ | Admitad |
| **Aviasales** | Покупка авиабилетов | CPS до 5% | Admitad |
| **Skillbox** | Оплата курса | CPA до 15 000₽ | Admitad |
| **AliExpress Россия** | Покупка товара | CPS 3-8% | Admitad |
| **МТС** | Подключение SIM / покупка | CPA 200-500₽ | Admitad |
| **Спортмастер** | Покупка товара | CPS до 10% | Admitad |
| **Альфа-Банк** | Заявка на карту / кредит | CPA до 3 000₽ | Admitad |
| **МФО (Moneyman, Lime и др.)** | Выдача займа | CPA 300-1 500₽ | Admitad |

### Как начисляются выплаты
1. Пользователь совершает целевое действие по ссылке представителя
2. Система фиксирует конверсию (статус: `pending`)
3. После подтверждения рекламодателем (статус: `approved`):
   - Среды зачисляются на баланс представителя (80%)
   - Среды зачисляются на баланс амбассадора (10%)
   - Средства зачисляются на баланс платформы (10%)
4. Представитель и амбассадор могут вывести средства

### Сроки подтверждения
- Финансовые офферы (банки, МФО): 1–30 дней (зависит от одобрения)
- E-commerce (Wildberries, Ozon): 15–45 дней (срок возврата товара)
- EdTech (Skillbox): 30 дней (период охлаждения)
- Рефералы (Яндекс Маркет): после выполнения условий (продажа от 15 000₽)

## 6. Правила для представителя

### Права
- Доступ ко всем активным офферам в каталоге
- Генерация персональных ссылок для каждого оффера
- Создание промокодов для приглашения новых участников
- Просмотр статистики конверсий и дохода
- Вывод средств

### Обязанности
- Соблюдать правила рекламы (маркировка, честность)
- Не использовать спам, ботов, накрутки
- Не создавать несколько аккаунтов

### Ограничения
- Один аккаунт на человека
- Вывод от 5 000₽
- Комиссия платформы за вывод: ~1% (комиссия банка за автоматический перевод)

---

## 7. Правила для амбассадора

### Как стать амбассадором
- По приглашению основателя
- Или автоматически: приведи 5+ активных представителей (каждый получил выплаты более двух раз)

### Права амбассадора
- Всё, что у представителя
- Дашборд команды: список представителей, их конверсии
- Повышенный процент (10%) с команды
- Приоритетная поддержка

### Обязанности
- Помогать представителям разобраться с платформой
- Не переманивать представителей у других амбассадоров
- Соблюдать кодекс партнёра

## 9. Запрещённые действия

- Спам (массовая рассылка, нецелевой трафик)
- Использование ботов, эмуляторов, накруток
- Создание нескольких аккаунтов
- Самостоятельное совершение целевых действий (покупка своих товаров)
- Размещение ссылок на сайтах 18+, экстремистского содержания
- Использование чужих торговых марок без разрешения

Нарушение → блокировка аккаунта без выплаты.

---

## 10. Общие положения

- Доход участников формируется исключительно от реальных действий пользователей (покупки, заявки, регистрации)
- Вознаграждение за приглашение новых участников выплачивается только с их активных конверсий
- Платформа оставляет за собой право менять ставки и правила с уведомлением за 7 дней
- Все налоговые обязательства участники несут самостоятельно

```

  - email, name, avatar, createdAt
```

### Позиционирование Hype

**Hype — платформа, объединяющая CPA-офферы и партнёрскую структуру с автоматическими отчислениями.**

| Критерий | Конкуренты | Hype |
|---|---|---|
| **Многоуровневые комиссии** | Нет (только 1 уровень) | 2 уровня: 80% → 10% → 10% |
| **Порог входа** | Нужен трафик / аудитория | Нулевой |
| **Российские бренды** | Выборочно | Wildberries, Ozon, Халва, Яндекс Маркет |
| **Промокоды** | Есть, без привязки | Промокод = привязка к структуре |
| **Автоматические отчисления** | Нет | Автомат: платформа → уровни |

### Преимущества

1. **Пустая ниша** — CPA + партнёрская структура
2. **Низкий порог входа** — не нужны подписчики, трафик, опыт
3. **Российские топ-бренды** — Wildberries, Ozon, Халва, Тинькофф
4. **Автоматизация** — вся цепочка от промокода до выплаты
5. **Заработок через команду** — без своего трафика

### Риски

1. **Зависимость от рекламодателей** — уход ключевых брендов ослабит предложение
2. **Качество трафика** — при массовом наборе конверсии могут быть низкими
3. **Техническая сложность** — трекинг многоуровневых комиссий требует надёжной архитектуры
